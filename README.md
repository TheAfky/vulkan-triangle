- [cimgui.zig](https://github.com/tiawl/cimgui.zig)
## Build
- You need to have Zig and glslc installed
### Linux
- zig build -Dtarget=x86_64-linux-gnu -Doptimize=ReleaseFast
### Windows
- zig build -Dtarget=x86_64-windows -Doptimize=ReleaseFast
