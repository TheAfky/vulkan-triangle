# Vulkan Triangle
- Written using Zig programming language.
- Made by TheAfky
## Libraries
- [zglfw](https://github.com/IridescenceTech/zglfw)
- [glfw.zig](https://github.com/tiawl/glfw.zig)
- [vulkan-zig](https://github.com/Snektron/vulkan-zig)
## Build
- You need to have Zig installed.
### Linux
- zig build -Dtarget=x86_64-linux-gnu
### Windows
- zig build -Dtarget=x86_64-windows
